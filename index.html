About Me

My journey into cybersecurity and cloud computing began with a fascination for secure system design and the ever-evolving landscape of digital threats. I am passionate about building robust, secure systems and protecting digital infrastructure from emerging vulnerabilities.

My interests span across multiple domains of cybersecurity and cloud engineering:

  • Secure System Design – Architecting systems with security as a foundational principle
  • SOC Alert Monitoring & Incident Response – Real-time threat detection and rapid response workflows
  • Cryptography – Understanding encryption, hashing, and cryptographic protocols
  • Cloud Fundamentals – Designing scalable, secure cloud architectures
  • Applied AI for Security – Leveraging machine learning for threat detection and security automation

As a Python and Java developer, I combine strong programming skills with security expertise to create solutions that are both functional and secure. My approach is disciplined, methodical, and always focused on continuous learning.
